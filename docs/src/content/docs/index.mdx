## Quick Install

1. Download the latest `.mrpack` from [GitHub Releases](https://github.com/lucasilverentand/lumo-optimized/releases)
2. Open Prism Launcher → Add Instance → Import from zip
3. Select the `.mrpack` file and click OK
4. Launch and enjoy!
