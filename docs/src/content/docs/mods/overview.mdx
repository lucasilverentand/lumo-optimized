## Mod Configuration

All mods can be configured through **Mod Menu**:

1. Press **Escape** to open the pause menu
2. Click **Mods**
3. Find the mod you want to configure
4. Click the settings icon (gear) next to it

Most mods also respect the pre-configured defaults from YOSBR, so you don't need to change anything unless you want to customize your experience.
